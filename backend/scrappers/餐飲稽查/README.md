# 餐飲業者資料爬蟲

從 [臺北市政府衛生局食藥粧網路地圖](https://imap.health.gov.tw/) 抓取餐飲業者資料的爬蟲程式。

## 功能特色

- 自動抓取多頁餐飲業者資料
- 支援 CSV 和 JSON 兩種輸出格式
- 包含業者名稱、地址、電話、登錄字號、合格狀態、經緯度座標

## 使用方法

```bash
python3 scrape_food_data.py
```

## 輸出檔案

程式會產生兩個檔案：

1. **food_data.csv** - CSV 格式，適合用 Excel 或試算表軟體開啟
2. **food_data.json** - JSON 格式，適合程式處理

## 系統需求

- Python 3.6+
- requests
- beautifulsoup4

安裝依賴套件：

```bash
pip3 install requests beautifulsoup4
```

## 注意事項

- 程式預設抓取前 5 頁資料
- 每頁約包含 1-2 萬筆資料（網站會回傳所有資料）
- 兩次請求之間會延遲 1 秒，避免對伺服器造成負擔
- 如遇連線問題，程式會繼續執行並顯示錯誤訊息
